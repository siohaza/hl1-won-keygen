# Half-Life 1 WON keygen

## Building

Install build dependencies.

```
sudo apt install cargo rust    # Debian
sudo dnf install cargo rust    # Fedora
sudo pacman -S cargo rust      # Arch Linux
```

Then build the program with:

```
cargo build
```

Run the tests with:

```
cargo test
```
